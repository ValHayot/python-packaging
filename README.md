# Colour My Brain (cmb)

A really cool command-line tool that creates a coloured png out of an anatomical NIfTI image.

## Installation

Use [pip](https://pip.pypa.io/en/stable) to install cmb

```bash
pip install cmb
```

## Usage

```bash
cmb input.nii coloured_brain.png
```
